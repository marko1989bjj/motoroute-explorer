# MotoRoute Explorer

MotoRoute Explorer is a responsive Vue 3 web application for discovering, filtering and saving motorcycle routes.

## Features

- Motorcycle route list
- Route filtering
- Dynamic route detail pages
- Favorite routes with Local Storage
- Responsive design for mobile, tablet and desktop
- Basic SEO setup

## Technologies

- Vue 3
- Vite
- Vue Router
- Pinia
- CSS3
- Local Storage

## Run locally

```bash
npm install
npm run dev
```
